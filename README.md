# Система отслеживания проникновения

## Установка
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

## Подготовка
Создайте папку data и поместите test.mp4 в неё:
mkdir -p data
cp /путь/к/test.mp4 data/

## Запуск
python main.py

## Использование
При первом запуске откроется окно разметки зоны.
Кликайте левой кнопкой мыши, чтобы добавить точки (минимум 3).
Нажмите 's' для сохранения, 'q' для выхода.
После разметки начнётся обработка видео.
Нажмите 'q' в окне видео для остановки.
